Conditional Statement

A conditional simply returns a value based on a true/false statement.

Example:

If the number is equal to 21, print “You win!”

If the number is greater than 21, print “BUST!”

Let’s look at the anatomy of an if/else statement:

if(condition) {
    // Do something here
 } else {
    // Do something else here
 }
if(score > 10) {
    document.write('You won!');
 } else {
    document.write('Game over!');
 }
