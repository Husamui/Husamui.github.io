Transition

Transitions enable you to define the transition between two states of an element

p{
    transition: property duration mode delay;
}
p{
    transition: color 2s ease-in-out 1s;
}

Transition Demo

#box {
	width: 200px;
	height: 200px;
	margin: 100px auto;
	background: #222;
	transition: all 0.3s ease;
}
#box:hover {
	width: 250px;
	height: 250px;
	background: #f00;
	border-radius: 20px;
}

Transform 2D

  • Translate
    Moving an element along the Y-axis X-axis

  • Rotate
    Rotates an element around a fixed point on the 2D plane

  • Scale
    Resizes an element on the 2D plane

  • Skew
    Skews an element on the 2D plane

Transform Translate Demo

#box {
	width: 200px;
	height: 200px;
	margin: 100px auto;
	background: #222;
	transition: all 0.3s ease;
}
#box:hover {
	transform: translate(30px, 30px);
}

Transform Rotate Demo

#box {
	width: 200px;
	height: 200px;
	margin: 100px auto;
	background: #222;
	transition: all 0.3s ease;
}
#box:hover {
	transform: rotate(45deg);
}

Transform scale Demo

#box {
	width: 200px;
	height: 200px;
	margin: 100px auto;
	background: #222;
	transition: all 0.3s ease;
}
#box:hover {
	transform: scale(1.5);
}

Transform skew Demo

#box {
	width: 200px;
	height: 200px;
	margin: 100px auto;
	background: #222;
	transition: all 0.3s ease;
